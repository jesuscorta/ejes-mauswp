## Objetivo de esta carpeta

- Explicar la arquitectura real del theme, no una intención antigua.
- Dejar claros los plugins y dependencias implícitas.
- Documentar qué piezas están resueltas por bloques ACF.
- Señalar decisiones recientes, limitaciones y zonas sensibles.
- Reducir el coste de cambio de modelo o de agente.
